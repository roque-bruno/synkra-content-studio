# Story 6.5 — Auditoria QA Completa — Zero Dados Falsos

**Epico:** 6 — Transparencia de Producao
**Status:** Ready
**Executor:** @qa (Quinn)
**Modo:** YOLO
**Dependencia:** 6.1, 6.2, 6.3, 6.4

---

## Descricao

Apos todas as correcoes de transparencia, executar auditoria QA completa da interface para garantir que NENHUMA informacao falsa, simulada ou enganosa restou na plataforma.

## Acceptance Criteria

- [ ] Todas as paginas verificadas — nenhum dado falso encontrado
- [ ] Status de plataformas reflete realidade (nao configurado quando nao ha credenciais)
- [ ] Banner de preview mode aparece corretamente quando sem API key
- [ ] Botao de publicar desabilitado quando sem plataforma conectada
- [ ] Toast de publicacao indica preview quando em preview mode
- [ ] Pagina de dashboard nao mostra metricas inventadas
- [ ] Health check reflete estado real de todas as dependencias
- [ ] Relatorio de auditoria gerado com todos os itens verificados

## Scope

**IN:**
- Auditoria visual de TODAS as paginas da interface
- Verificacao de cada endpoint que alimenta dados visuais
- Teste em preview mode (sem API key) e production mode (com API key)
- Relatorio formal de auditoria

**OUT:**
- Correcoes de bugs (abrir stories separadas se encontrar)
- Testes automatizados (out of scope para este epic)

## Checklist de Auditoria

### Pagina: Dashboard
- [ ] Metricas mostram dados reais ou "Sem dados" (nunca numeros inventados)
- [ ] Graficos vazios quando sem historico
- [ ] Status geral reflete preview mode quando aplicavel

### Pagina: Pipeline
- [ ] Pecas mostradas sao reais (do banco/output)
- [ ] Status de cada peca e real
- [ ] Nenhum conteudo placeholder apresentado como real

### Pagina: Publicar
- [ ] Status de plataformas reflete configuracao real
- [ ] Botao publicar desabilitado quando sem plataformas
- [ ] Modal de publicacao filtra plataformas nao configuradas

### Pagina: Settings
- [ ] Status de API keys mostra configurado/nao configurado real
- [ ] Health bar reflete testes reais

### Pagina: Calendario
- [ ] Eventos mostrados sao reais (do banco)
- [ ] Sem eventos placeholder/demo

### Pagina: Relatorios
- [ ] Dados de relatorio sao reais ou "Sem dados"
- [ ] Sem metricas fake
