# Story 7.5 — Feedback Visual de Status de Integracao

## Descricao

Unificar o feedback visual de status de integracoes em toda a plataforma. O status configurado na pagina de Settings deve refletir automaticamente no dashboard, na pagina de publicacao e em qualquer outro lugar que mostre estado de plataformas.

## Acceptance Criteria

- [ ] Status de integracao consistente em TODAS as paginas (Settings, Publicar, Dashboard)
- [ ] Icone e cor padrao para cada estado: configurado+conectado (verde), configurado+erro (vermelho), nao configurado (cinza)
- [ ] Pagina de Settings mostra ultimo teste e latencia
- [ ] Dashboard mostra resumo: "3 de 5 integracoes ativas"
- [ ] Pagina de Publicar mostra apenas plataformas conectadas como opcao
- [ ] Transicao visual animada quando status muda (salvar key → testar → conectado)

## Scope

**IN:**
- Padronizacao de icones/cores de status em toda a UI
- Componente Alpine.js reutilizavel para status de plataforma
- Integracao cruzada entre paginas

**OUT:**
- Notificacoes push de mudanca de status
- Historico de testes

## Tasks

1. Criar funcao helper `platformStatusClass(status)` retornando classes CSS
2. Criar funcao helper `platformStatusIcon(status)` retornando icone SVG
3. Aplicar em Settings, Publicar e Dashboard
4. Mostrar resumo de integracoes no Dashboard
5. Animacao de transicao de status
